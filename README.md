# NazRadio

MPD client on nodejs, with responsive design. 

## Libraries

### Front-end
* BackboneJS
* jQuery
* Bootstrap
* RequireJS

### Back-end
* MPD
* expressJS
* socket.io
* underscore


## Installation

```sh
git clone https://github.com/nazstone/NazRadio.git nazradio
cd nazradio
sudo npm install
```
## Configuration
Edit the javascript file `conf / config.js`

### Radio
A interface to quickly add/remove radio is available
