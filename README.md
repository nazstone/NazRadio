# NazRadio

MPD client on nodejs, with responsive design. 

## Libraries

### Front-end
* BackboneJS
* jQuery
* Bootstrap
* RequireJS

### Back-end
* MPD
* expressJS
* socket.io
* underscore


## Installation

```sh
git clone https://github.com/nazstone/NazRadio.git nazradio
cd nazradio
sudo npm install
```
